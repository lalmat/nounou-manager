# Nounou Manager

## Description
This project aims to track you nurse hours. It can be used in France to help to make you PAGE declaration

## Installation
* Clone the projet
* `$ composer install`
* `$ npm install`
* Configure .env file
* `$ php artisan migrate`

## Start to develop
Launch 2 terminals :
- `php artisan serve`
- `npm run hot`

Then goto the localhost:8000 page

## Technologies
- Laravel 6
- VueJS 2
